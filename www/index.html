🛰️ Set up 100% Reliable Background Tracking (OwnTracks)

For bulletproof tracking when your phone is locked, minimized, or in your pocket:

  1. Install the free OwnTracks app from the iOS App Store or Google Play Store.
  2. Open OwnTracks, tap the menu (cog icon), and select Connection.

🔋 Reduce OwnTracks Battery Consumption:

  • Use Significant Changes Mode (Recommended): On the main OwnTracks screen, tap the monitoring mode button (it looks like a small play/stop button or arrows) and switch it from "Move mode" to "Significant changes". This uses network/cell-towers and updates every ~500m, consuming almost 0% battery.
  • Tweak Locator Settings: If using Move mode, go to settings (cog icon) → Locator. Set Displacement to at least 150 (meters) and Interval to 300 (seconds). This stops continuous GPS scanning.
  • Set Battery to Unrestricted: In Android settings, search for OwnTracks and set battery usage to "Unrestricted" to allow it to run efficiently in the background without Android forcing it to wake up aggressively.
📱 Phone Settings for Continuous Background GPS

To prevent iOS or Android from putting the tracking to sleep when the screen is locked or the app is minimized:

  • Android: Go to Settings → Apps → Temple Tour Tracker → Permissions → Location. Select "Allow all the time". Also disable "Pause app activity if unused" and set Battery to "Unrestricted".
  • iOS: Go to Settings → Privacy & Security → Location Services → Temple Tour Tracker. Select "Always" and ensure "Precise Location" is turned ON.
  • Battery: Turn off Power Saving / Battery Saver mode, as it blocks background GPS scanning.

For native apps, native background services and boot receivers will automatically recover if suspended.
